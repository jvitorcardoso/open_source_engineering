# Open Source Engineering

> Airflow, Spark, dbt e tudo o que há de bom.

Este repositório é dedicado ao desenvolvimento e compartilhamento de conhecimento sobre engenharia de dados open source, focando nas melhores ferramentas e práticas da área.

## 📋 Sobre o Projeto

O **Open Source Engineering** é um hub centralizado para recursos, exemplos e documentação relacionados às principais tecnologias de engenharia de dados open source. O objetivo é fornecer um ponto de partida robusto para profissionais que trabalham com pipelines de dados modernos.

## 🚀 Tecnologias

Este projeto explora e documenta as seguintes tecnologias:

### Apache Airflow
- **Orquestração de workflows**: Gestão e automação de pipelines de dados
- **DAGs (Directed Acyclic Graphs)**: Definição de fluxos de trabalho complexos
- **Monitoramento**: Acompanhamento de execuções e alertas

### Apache Spark
- **Processamento distribuído**: Análise de grandes volumes de dados
- **APIs múltiplas**: Suporte para Scala, Python, R e SQL
- **Machine Learning**: MLlib para algoritmos de aprendizado de máquina

### dbt (data build tool)
- **Transformação de dados**: Modelos SQL para transformações
- **Testes automatizados**: Validação de qualidade dos dados
- **Documentação**: Geração automática de documentação dos modelos
